"# LineBot_pttmovies"
====
本程式用途為輸入電影名稱關鍵字後，利用該關鍵字在 PTT_Movie 網頁板進行搜尋，抓取最新 10 頁資料，分別統計好雷、普雷及負雷數目進行分析，得知該電影在  PTT_Movie 板之評價。

How to use
----
LINE 好友搜尋 ID：@vsr0046b，名稱為「鄉民怎麼看」，加入好友即可使用。
